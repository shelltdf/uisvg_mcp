# 用户说明书：UISVG 编辑器

## 环境前提

- 现代浏览器（Chrome / Edge / Firefox 等）。
- 本地运行开发服务器或通过静态托管打开构建产物。

## 启动（开发）

在仓库 `uisvg-editor/` 目录执行依赖安装与开发服务（详见 [developer-manual.md](./developer-manual.md)），浏览器访问终端提示的本地 URL。

## 基本操作

- **帮助 → UISVG 规范与类型…**：打开居中悬浮窗口，列出命名空间 URI、**全部 uisvg 扩展类型**（QName、语义 localName、与组件库一致的大类）；**每种类型可展开**，查看与右栏「UI 属性」一致的 **语义属性**（`data-uisvg-ui-props` 键名、说明、编辑方式及枚举/掩码补充）；并附 **文档结构 / 语义标签与 QName / 嵌套层级** 三段关系说明。
- **新建**：菜单或工具栏「新建」创建空白 `*.ui.svg` 草案。
- **打开**：选择本地 `.ui.svg` 文件。
- **保存**：将当前设计下载为 `.ui.svg`。
- **画布**：逻辑画布区域由编辑器绘制（白底与网格）；左上角显示宽高、DPI、原点 (0,0) 等只读信息（不写入保存的 SVG）。**视图 → 画布设置…** 可调整画布宽高、网格步长、DPI。滚轮缩放；**中键拖拽**平移视图；**左键**点选 **UISVG 对象**（对象根 `<g>` 或 `#layer-root`）；**Shift+左键** 多选；**空白处拖拽** 框选多选。与左侧 **UI 大纲**、右侧属性联动。选中后出现虚线框（多选为各对象框的并集）；**在已选中对象上按住左键拖拽**可移动，靠近对齐位置时出现红色对齐线并吸附。在 **左下角对齐调试** 面板内滚动鼠标滚轮时不会缩放画布。
- **多选与对齐**：至少选中 **2 个**非文档根的可移动对象后，可用 **工具栏六个对齐图标**、**菜单「编辑」→ 对齐** 或 **画布右键菜单 → 对齐** 执行左/右/水平居中/顶/底/垂直居中。画布右键菜单顺序为：**复制**、**粘贴**、**对齐**（子菜单）、**删除**（最后一项）。未满足条件时工具栏对齐按钮为灰色不可用。
- **对齐调试**：**视图** 或 **工具栏调试图标** 可显示/隐藏左下角面板；面板 **仅显示最近一次对齐** 的摘要（并集、`d` 等），不混排选中框几何与框选日志。
- **UI 大纲**：根为 **uisvg::root**（对应画布内容根 `#layer-root`）；下列仅 **对象根 `<g>`**（不单独列出 uisvg 类型子节点）。表头两列：**标识（DOM `id`）**、**UISVG 类型**；行内第一列为各对象的 `id`、第二列为类型全称。两列之间可 **拖动竖向分隔条** 调整列宽比例（约 25%～75%），设置会保存在本机浏览器；过长文本行内省略，悬停行可见完整说明。表头与属性区文案随 **设置 → 语言** 切换（中/英）。
- **组件库 / 属性**：左侧 dock（大纲与组件库可拖分割条）；右侧属性区（可拖分割条）。各分区标题旁 **◀ / ▶** 收起到侧条；**▼** 垂直折叠内容。**Windows** 类控件可 **单击** 按占位算法插入画布，也可 **按住拖到画布**：释放在 **Form / Panel / GroupBox** 等容器上时，新控件会作为该容器的 **子节点** 插入（落点约在指针处，并限制在容器客户区内）；释放在空白处则与单击类似，插入在内容根下。

## 文件

- 扩展名：**`*.ui.svg`**
- 可用支持 SVG 的软件打开；**UISVG** 语义在命名空间子节点与属性中（参见 `00-concept/uisvg-format-spec.md`）；纯查看器至少可渲染标准 SVG 几何。
