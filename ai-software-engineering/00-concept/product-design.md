# 产品设计：UISVG 编辑器

## 愿景

提供一款在浏览器中运行的 UI 设计工具，**视觉与交互习惯贴近 Windows 桌面窗口**（菜单栏、工具栏、可拖拽分割的侧栏、状态栏），中心为**可缩放、平移的网格画布**，用于编辑基于 **SVG** 的 **uisvg** 设计稿。设计稿以 `*.ui.svg` 保存，可被任意支持 SVG 的软件打开预览（兼容部分取决于查看器对元数据的保留策略）。

## 目标用户

- 需要在矢量画布上规划界面结构与组件占位的产品/设计/开发人员。

## 主窗口布局

1. **顶部**：菜单栏（文件、编辑、视图、帮助等）。
2. **菜单栏下方**：工具栏（常用操作快捷按钮）。
3. **左侧（垂直分割）**：
   - **上**：UI 大纲树（文档层级结构）。
   - **下**：UI 组件库（可拖拽或点击添加到画布的组件条目）。
4. **中心**：画布区域——带**背景网格**，支持鼠标**缩放**与**平移**。
5. **右侧**：**属性** 面板（选中项的属性、文档元数据或源码片段等；与画布之间可拖拽调整宽度）。
6. **底部**：状态栏（光标、缩放比例、提示信息）。

### 编辑器外壳与画布（交互要点）

- **工具栏**：除常规文件与视图操作外，提供 **六个对齐** 图标（左/右/水平居中/顶/底/垂直居中）；图标 **始终可见**，当选中可移动对象不足 **2** 个时为禁用态。**调试** 开关控制画布 **左下角** 是否显示 **对齐调试** 摘要（仅反映最近一次对齐，不展示选中框/框选等冗长日志）。
- **画布**：支持 **多选**（Shift 点选、空白处框选）、**拖拽移动** 与 **吸附对齐线**；对齐也可通过 **编辑** 菜单与 **画布右键菜单** 执行。
- **UI 大纲**：表头为 **标识（DOM `id`）** 与 **UISVG 类型** 两列，列间可 **拖动竖条** 调整比例（约 25%～75%），设置保存在本机浏览器；界面语言可在设置中切换。

## 核心用例

| 用例 | 说明 |
|------|------|
| 新建/打开/保存 | 打开与保存 `*.ui.svg` |
| 画布导航 | 缩放、平移视图 |
| 大纲浏览 | 查看与选择文档节点 |
| 组件使用 | 从组件库向画布添加占位组件 |

## 成功标准

- 界面分区清晰，与 Windows 常见 MDI/编辑器窗口风格一致。
- 画布网格与变换流畅；文件可被标准 SVG 查看器打开（可见 SVG 内容）。
