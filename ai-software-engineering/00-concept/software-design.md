# 软件设计：UISVG 编辑器

## 概述

实现形态为 **Vue 3** 单页应用（SPA），构建目标见 `02-physical/uisvg-editor-web/`。应用划分为：

- **Shell**：菜单栏、工具栏、状态栏、三栏布局容器（左/中/右）。
- **大纲模块**：树形展示文档节点（与画布 DOM/SVG 结构或逻辑模型同步）。
- **组件库模块**：静态或配置驱动的组件列表，与画布插入逻辑对接。
- **画布模块**：SVG（或嵌套 SVG）视口、网格背景、视图变换（平移/缩放）、命中测试与选择（后续迭代）。
- **属性面板**：展示选中节点属性、原始片段或 JSON（只读/后续可编辑）。
- **文档模型**：内存中的 SVG / uisvg 解析结果；保存时序列化为 `*.ui.svg`。

## 与格式规范关系

- 磁盘格式以 [uisvg-format-spec.md](./uisvg-format-spec.md) 为准；解析与序列化逻辑须与之对齐。
